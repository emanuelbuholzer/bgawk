## building

- install rust: https://www.rust-lang.org/tools/install
- on linux you need `libjack-dev` and `libasound2-dev` (`jack-devel` and `alsa-lib-devel` on void)
- clone
```
git clone https://github.com/tomara-x/bgawk.git
```
- build
```
cd bgawk
cargo run --release
```
